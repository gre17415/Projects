# Echo

В этой задаче нужно реализовать echo server.
Сервер должен принимать запрос и возвращает его обратно.

Для взаимодействия пользовательского кода с сервером будем использовать две [очереди](https://userver.tech/d0/d92/classconcurrent_1_1MpscQueue.html). Через одну сервер должен получать сообщения и отправлять их через другую.

Ваша реализация должна уметь обслуживать одновременно несколько клиентов. Для этого можно на каждое соединение запускать новый файбер.

Реализуйте классы `Channel` и `EchoClient`.
Класс `Channel` предназначен для прямого общения с сервером через очереди. `EchoClient` обертка над `Channel`.

Так же нужно реализовать и серверную часть в функции `ServeForever`. На вход подается очередь в которую клиенты отправляют запросы (`Connection`) при создании.

Можно модифицировать файлы state.h, echo.cpp.
